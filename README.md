# Xamarin.Forms UVENTO
O que foi usado no desenvolvimento:
- Padrão MVVM (sem framework)
- Embedded Fonts (https://docs.microsoft.com/en-us/xamarin/xamarin-forms/user-interface/text/fonts)
- CollectionView
- XamarinForms 5.0
- Xamarin Community Toolkit
